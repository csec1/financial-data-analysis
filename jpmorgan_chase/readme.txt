Overview
This Python script analyzes a dataset of financial transactions to extract insights and visualize patterns, particularly focusing on fraud detection.

Features
Data Exploration: Loads and explores a CSV dataset, displaying basic info and statistical summaries.
Query 1 - Transaction Type Distribution: Displays the count of each transaction type.
Query 2 - Fraud Analysis: Shows the number of flagged and actual fraudulent transactions.
Query 3 - Transaction Volume by Type: Summarizes the total transaction volume for each type.
Data Visualization:
Bar charts for transaction type distribution.
Comparison of flagged vs. actual fraud cases.
Total transaction volume by type.
Requirements
Python 3.x
pandas, matplotlib, and seaborn libraries
Install dependencies using:

bash
Copy code
pip install pandas matplotlib seaborn
Usage
Run the script:

bash
Copy code
python3 main.py
The script prints the analysis results to the console and generates charts for visual insights.

Output
Console: Displays dataset info, statistical summary, transaction counts, fraud analysis, and transaction volumes.
Charts: Visualizes transaction type distribution, fraud comparison, and transaction volumes by type.
